## storyboard

```
🎬 35초 유튜브 쇼츠 스토리보드

주제: 피부를 망치는 5가지 습관

0~3초 (인트로)

영상: 밝은 배경, 남녀 캐릭터가 거울을 보며 “피부 고민” 표정.

텍스트/자막: “오늘은 피부를 망치는 5가지 습관!”

나레이션: “남녀 모두 주의해야 할 피부 파괴 습관, 바로 알려드립니다!”

3~9초 (습관 1. 과도한 클렌징)

영상: 사람이 세수를 뽀득뽀득 하면서 피부가 갈라지는 애니메이션 효과.

자막: “❌ 과도한 세안 → 피부 장벽 손상”

나레이션: “뽀득 소리 날 때까지 세안? 피부 장벽만 무너집니다.”

컷 전환: 약산성 클렌저 + 부드럽게 물로 헹구는 장면.

자막: “✅ 하루 2번, 약산성 클렌저!”

9~15초 (습관 2. 불충분한 수면)

영상: 밤새 스마트폰 보는 모습 → 얼굴 칙칙, 다크서클 과장 연출.

자막: “❌ 수면 부족 → 탄력 저하, 다크서클”

나레이션: “잠이 부족하면 피부가 늙습니다.”

컷 전환: 푹 자는 모습 + 달력에 7~8시간 체크.

자막: “✅ 하루 7~8시간 숙면!”

15~21초 (습관 3. 잘못된 식습관)

영상: 햄버거·콜라 → 여드름 얼굴 등장.

자막: “❌ 단 음식, 가공식품 → 피부 트러블”

나레이션: “달고 기름진 음식은 여드름 유발자!”

컷 전환: 신선한 채소, 과일, 물 마시는 장면.

자막: “✅ 신선한 음식 + 하루 2L 물!”

21~27초 (습관 4. 자외선 차단제 미사용)

영상: 흐린 날씨에도 자외선이 피부에 닿아 주름과 기미가 생기는 애니메이션.

자막: “❌ 선크림 미사용 → 노화 촉진”

나레이션: “흐린 날에도 자외선은 존재합니다.”

컷 전환: SPF 30 자외선 차단제 바르는 장면.

자막: “✅ 매일 자외선 차단제!”

27~31초 (습관 5. 무의식적 접촉)

영상: 턱 괴기, 손으로 얼굴 만지기, 휴대폰에 세균 번쩍 효과.

자막: “❌ 얼굴 만지기 → 세균 전이”

나레이션: “손은 세균 천국! 피부 트러블 직행.”

컷 전환: 손 씻는 장면, 휴대폰 닦는 장면.

자막: “✅ 손 청결 + 휴대폰 클리닝”

31~35초 (마무리)

영상: 앞에서 등장한 남녀 캐릭터가 환하게 웃으며 맑은 피부 표현.

자막: “작은 습관만 고쳐도 피부는 달라집니다✨”

나레이션: “오늘부터 실천하세요. 건강한 피부는 꾸준함의 선물입니다.”

👉 이렇게 35초 안에 빠른 컷 전환 + 시각적 대비 + 해결책 강조로 구성하면 시청자가 끝까지 몰입하게 됩니다.

```
